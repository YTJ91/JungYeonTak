CSS 글씨 다루기

  • font-size

    글씨의 크기를 지정한다.

  • font-weight

    글씨의 가중도(굵기)를 지정한다.100~900숫자나 bold lighter 등으로 지정한다

  • text-allgn

    글씨의 정렬방법을 지정한다

    justify는 양쪽 정렬, lorem숫자 입력 시 숫자 만큼 임의의 명단이 나옴

  • line-height

    줄의 높이를 지정한다.

    Lorem ipsum dolor sit amet consectetur adipisicing elit. Dolorum, sapiente?

  • word-spacing, letter-spacing

    단어, 글자 간의 간격을 지정한다

  • text-transform

    단어를 변형시킨다.

    lowcase는 소문자로 uppercase는 대문자로

  • text-decoration

    단어를 밑줄, 취소선 등으로 꾸민다
