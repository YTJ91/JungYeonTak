CSS 선택자

CSS 선택자(Selector)란 스타일을 적용하고자 하는 요소를 골라내는 태그

선택자의 종류

  • 전체 선택자(*)

    전체 선택자는 html 문서 내의 모든 요소에 적용한다 우선 점수는 0점이다.

  • 타입(태그) 선택자

    요소 타입(태그명)을 통해서 요소를 선택한다. 문서 내에 있는 해당하는 타입의 요소를 모두 선택한다. 우선 순위 점수가 1점이다

  • 클래스 선택자(.)

    특정 클래스 값을 가진 요소를 모두 선택한다. 클래스 값은 요소의 class 속성을 통해서 지정한다.
    띄어쓰기를 통해서 여러 개의 클래스를 지정할 수 있다.
    같은 클래스 값을 여러 요소가 가질 수 있다.
    우선 순위 점수는 10점이다.

  • 아이디 선택자(#)

    특정 아이디값을 가진 요소를 선택한다. 아이디 값은 요소의 id 속성을 통해서 지정한다.
    아이디 값은 여러개를 지정할 수 없다.
    문서 내의 아이디 값은 유일해야한다.
    우선 순위 점수는 100점이다.

  • 일치 선택자

    여러 선택자에 모두 일치하는 요소만을 선택한다.

  • 자식 선택자(>)

    특정 요소의 자식 요소만을 선택할 수 있다

  • 후손 선택자

    특정 요소의 후손 요소를 모두선택할 수 있다. 띄어쓰기를 통해서 구분한다.

  • 인접 형제 선택자(+)

    A와 B가 같은 계층에 있을 때 A 바로 뒤에 B를 선택자로 지정합니다.

  • 일반 형제 선택자(~)

    A와 B가 같은 계층에 있을 때 A 뒤에 있는 모든 B를 선택자로 지정합니다.

  • 가상 클래스(:)

    특정 조건에 만족했을 때 (예: 요소에 마우스가 올라갔을 때) 가상의 클래스가 생긴 것 처럼 동작한다.

  • 가상 요소(::)

    HTML 문서에 포함되지 않았지만 디자인 적으로 필요할 때 가상요소를 만들어낸다.

상속과 우선 순위

글씨와 관련된 속성은 후손 요소에게도 상속된다.

같은 속성에서 값을 여러 개 적용하면 기본적으로 마지막에 적용된 값이 적용된다.
순서와 상관없이 우선 순위 점수 합산이 높은 선택자의 스타일이 적용된다.
인라인 방식은 우선 순위 점수가 1000점이기 때문에 주의해야한다.
!important를 사용하면 최우선으로 적용한다.
점수가 같을 경우에는 마지막에 작성한 코드가 적용된다.
